# FewPhotos

Photo gallery web application built with Flask.

Demo SQLite database with sample images is included.

## Features

- upload photos
- view gallery
- simple web interface
- SQLite database

## Technologies

- Python
- Flask
- HTML
- CSS
- SQLite
